# Curiosities (趣味物品)

一个为Minecraft 1.20.1添加多种有趣功能和物品的Forge模组。内容均为AI制作

## 版本信息

- 当前版本: 2025.5 (2025年5月)
- Minecraft: 1.20.1
- Forge: 47.2.20

## 主要功能

### 状态效果

- **狼群领袖**
    - 使周围30格内的狼/狗会协助攻击你锁定的目标
    - 同时会被熊猫敌视
    - 通过食用狼牙土豆获得此效果

- **罪恶**
    - 增加村民交易价格
    - 与村庄英雄效果相反
    - 通过道德天平附魔触发

### 附魔

- **连锁挖掘**
    - 使工具能够连锁挖掘相同类型的方块
    - 最高4级，等级越高可连锁的方块越多
    - 适用于各种挖掘工具

- **超级时运**
    - 效果是原版时运的1.5倍
    - 可与精准采集兼容（获得完整方块的同时增加掉落物）
    - 最高3级

- **道德天平**
    - 对和平生物造成双倍伤害，但会获得罪恶效果
    - 对敌对生物造成一半伤害，但掉落物增加3倍
    - 通过特殊配方合成获得

### 物品

- **狼牙土豆**
    - 食用后获得狼群领袖效果
    - 提供丰富的饥饿值和饱食度
    - 通过组合烤土豆、骨头和岩浆膏合成

- **火箭靴**
    - 允许玩家蹲下蓄力后跳得更高
    - 蓄力时间影响跳跃高度
    - 降落时减少摔落伤害
    - 使用燃料系统，可通过常见燃料补充

- **镰刀**
    - 多种材质可选：木质、石质、铁质、金质、钻石和下界合金
    - 一次收获大范围成熟作物并自动补种
    - 具有"丰收之舞"特性，有几率触发更大范围收获
    - 可作为近战武器使用

## 配置选项

每个功能模块都可以在配置文件中单独启用或禁用。可通过游戏内配置界面或直接编辑配置文件进行调整。

## 安装方法

1. 安装Minecraft Forge 1.20.1
2. 将模组文件放入Minecraft的mods文件夹
3. 启动游戏，尽情探索新增内容！

## 兼容性

- 支持Minecraft 1.20.1
- 与大多数其他模组兼容
- 可选择性集成JEI (Just Enough Items)，提供更好的配方展示

## 常见问题

**问：如何获得道德天平附魔？**  
答：这个附魔无法通过附魔台获得，只能通过特殊配方合成。

**问：为什么我的火箭靴不工作？**  
答：确保你有足够的燃料，并且正确使用（蹲下+长按跳跃键蓄力）。

**问：镰刀如何使用？**  
答：右键点击成熟的作物即可一次性收获范围内的所有成熟作物并自动补种。

## English Translation

### Curiosities

A Forge mod for Minecraft 1.20.1 that adds various fun features and items. All content is made by AI.

### Version Info

- Current Version: 2025.5 (May 2025)
- Minecraft: 1.20.1
- Forge: 47.2.20

### Main Features

#### Status Effects

- **Wolf Pack Leader**
    - Wolves/dogs within a 30-block radius will assist in attacking the player's targeted enemies
    - Makes pandas hostile toward the player
    - Obtained by consuming a Wolf Fang Potato

- **Guilt**
    - Increases villager trade prices
    - Opposite effect of Hero of the Village
    - Triggered by the Moral Balance enchantment

#### Enchantments

- **Chain Mining**
    - Allows tools to chain-mine blocks of the same type
    - Maximum level 4, higher levels can chain more blocks
    - Applicable to various digging tools

- **Super Fortune**
    - Provides 1.5x the effect of vanilla Fortune
    - Compatible with Silk Touch (can get both increased drops and intact blocks)
    - Maximum level 3

- **Moral Balance**
    - When attacking peaceful creatures: 2x weapon damage but player gets the "Guilt" debuff
    - When attacking hostile creatures: half damage but 3x drop multiplier
    - Only obtainable through special crafting recipes

#### Items

- **Wolf Fang Potato**
    - Grants the "Wolf Pack Leader" effect when consumed
    - Provides rich hunger and saturation values
    - Crafted by combining a baked potato, bones, and magma cream

- **Rocket Boots**
    - Allows players to crouch and charge jumps for higher jumps
    - Charge time affects jump height
    - Reduces fall damage when landing
    - Uses a fuel system that can be replenished with common fuels

- **Scythes**
    - Available in multiple materials: wooden, stone, iron, golden, diamond, and netherite
    - Harvest mature crops in bulk and automatically replant
    - Features a "Harvest Dance" function with a chance to trigger large-area harvesting
    - Can be used as melee weapons

### Configuration Options

Each feature module can be individually enabled or disabled in the configuration files. These settings can be adjusted
through the in-game configuration screen or by directly editing the config files.

### Installation

1. Install Minecraft Forge for 1.20.1
2. Place the mod file in your Minecraft mods folder
3. Start the game and enjoy the new content!

### Compatibility

- Supports Minecraft 1.20.1
- Compatible with most other mods
- Optional integration with JEI (Just Enough Items) for better recipe display

### FAQ

**Q: How do I get the Moral Balance enchantment?**  
A: This enchantment cannot be obtained through an enchanting table, only through special crafting recipes.

**Q: Why aren't my Rocket Boots working?**  
A: Make sure you have sufficient fuel and are using them correctly (crouch + hold jump key to charge).

**Q: How do I use the Scythe?**  
A: Right-click on mature crops to harvest all mature crops in range and automatically replant them.
